Mouse Events

Click Event

<div id="app">
	<button v-on:click="increase">Increase</button>
	<p>{{ counter }}</p>
</div>
new Vue({
	el: '#app',
	data: {
		counter: 0
	},
	methods: {
			increase: function(){
				this.counter++;
			}
	}
});

Mousemove Event

<div id="app">
	<p v-on:mousemove="updateCoordinates">Coordinates: {{ x }}/{{ y }}</p>
</div>
new Vue({
	el: '#app',
	data: {
		x: 0,
		y: 0,
	},
	methods: {
			updateCoordinates: function(event){
				this.x = event.clientX;
				this.y = event.clientY;
			}
	}
});

Mousemove Event with Modified Event in Child

This means that child element with not affected by the mouse move event.

<div id="app">
	<p v-on:mousemove="updateCoordinates">Coordinates: {{ x }}/{{ y }} <span v-on:mousemove.stop="">Stop Event<span/></p>
</div>

Properties

Computed Properties Vs Methods Properties

If you declared a rendering function in the page:

Computed will execute the function only if value of related data variable whithin that function is changed.

Method will always run every time data variable are changed. (Disadvantage if your functions code is too long script which is executed but not related on changes, it only interupt the speed performance of app)

Calling function:

Computed called without parenthesis

Method called with parenthesis

<div id="app">
		<button v-on:click="counter1++">Increase</button>
		<button v-on:click="counter1--">Decrease</button>
		<button v-on:click="counter2++">Increase Counter 2</button>
		<p>Counter 1: {{ counter1 }}</p>
		<p>Counter 2: {{ counter2 }}</p>
		<p>Result: {{ renderFlag }} | {{ renderCounter() }}</p>
	</div>
new Vue({
	el: '#app',
	data: {
			counter1: 0,
			counter2: 0,
			flag: ''
	},
	computed: {
			renderFlag: function(){
				console.log('Computed Triggered'); // will not triggere if you click the Increase Counter 2 button
				return this.counter1 > 5 ? 'Greater than 5' : 'Less than 5';
			}
	},
	methods: {
			renderCounter: function(){
				console.log('Method Triggered');
				return this.counter1;
			}
	}
});

Watch Properties

Will be executed when a specific data variable is changed.

new Vue({
	el: '#app',
	data: {
		counter: 0,
	},
	watch: {
		counter: function(){
			console.log('Watch Triggered');
		}
	},
});

Basic / Active & Innactive

A box that when you click, it change it state to active and inactive.

	width: 100px;
	height: 100px;
	margin: 10px;

<template> tag will not show in DOM but can use for conditional

Rendering List / For Loop

Let's assume that you have this arrays.

new Vue({
	el: '#app',
	data: {
			ingredients: ['meat','fruit', 'cookies'],
			persons: [
				{ name: 'Max', age: 27, color: 'red' },
				{ name: 'Anna', age: 'unknown', color: 'blue' }
			]
	},
});

Simple for loop

<div id="app">
	><ul>
		><li v-for="ingredient in ingredients" :key="ingredient">{{ ingredient }}></li>
	></ul>
></div>

Loop with index

<div id="app">
	<ul>
		<li v-for="(ingredient, index) in ingredients" :key="index">{{ index }}. {{ ingredient }}</li>
	</ul>
</div>

Loop with array of object

<div id="app">
	<ul>
		<li v-for="person in persons" :key="person.name">{{ person.name }}</li>
	</ul>
</div>

Loop of objects of array loop

will also loop the keys

<div id="app">
	<ul>
		<li v-for="person in persons" :key="person.name">
			<p v-for="(value, key, index) in person" :key="index">{{ index }}.) {{ key }}: {{ value }}</p>
		</li>
	</ul>
</div>

Lifecycle Method

  • beforeCreate()
  • created()
  • beforeMount()
  • beforeUpdate()
  • updated()
  • beforeDestroy()
  • destroy()
<div id="app">
	<p>{{ title }}</p>
	<button @click="handleUpdate">Change Title</button>
	<button @click="handleDestroy">Destroy</button>
</div>
new Vue({
	el: '#app',
	data: {
			title: 'test'
	},
	methods: {
			handleUpdate: function(){
				this.title = "changed title";
			},
			handleDestroy: function(){
				this.$destroy();
			}
	},
	beforeCreate: function(){
			console.log('beforeCreate')
	},
	created: function(){
			console.log('created')
	},
	beforeMount: function(){
			console.log('beforeMount')
	},
	beforeUpdate: function(){
			console.log('beforeUpdate')
	},
	updated: function(){
			console.log('updated')
	},
	beforeDestroy: function(){
			console.log('beforeDestroy')
	},
	destroy: function(){
			console.log('destroy')
	}
})
