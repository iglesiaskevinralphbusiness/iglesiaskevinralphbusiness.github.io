Git Command

Installation

Download git

Download cmder tool

Commands

Up 1 folder

cd ..

Open folder

cd myfolder

Show list of all folders and files

ls

Create new folder

mkdir myfolder

Remove / Delete

rm myfolder

Setup

Check Version

git --version

Config User

git config --global user.name myusername
git config --global user.email myemail

Check User Config

git config user.name
git config user.email

Exit on error

Type Shift + : + wp and hit enter

:wq

Create repository

git init

Staging Files

Show files in staging area

git status

Add files in staging area

git add .

Remove files in staging area

git rm --cached index.html

Commit

Commit files from staging area

git commit -m "your message here"

Show all commit

git log --oneline

Show all commit with descriptions

git log

Revert

Go or load to particular commit

git checkout commit_id

Go or load the lastest commit

git checkout master

Branch

Create new branch

git branch branch_name

Show all created branches

git branch -a

Open branch

git checkout 'branch_name'

Delete branch

Make sure that the branch you want to delete should not be opened in your terminal.

git branch -D branch_name

Merge branch

Make sure you are in master branch

git merge branch_name

Clone

git clone https://github.com/iamshaunjp/git-one.git

Push

Push no remote

This use when you don't have remote setup.

git push https://github.com/iamshaunjp/git-one.git master

Push with remote

You need to set up the remote

git remote add origin https://github.com/iamshaunjp/git-one.git
git push origin master

Push to custom branch

First, pull the repository, create new branch, do the edits, and commit and push the branch to git server.

git push origin branch_name

Others

Remove staging files

git reset HEAD

Remove staging file and checkout

git reset --hard
git merge branch_name  --no-ff

Stash

git stash
git stash show
git stash pop
git stash clear
git stash drop

Rename last unpushed commit message

git commit --amend -m "New commit message"

Remove last commit local

git reset --hard HEAD^

Remove all commit local

git reset --hard HEAD^2

Show difference before commit

git diff
	git diff --staged

Abort merging

git merge --abort

Rename branch

git branch -m new-name

Show all branch hidden

git branch -a

Squash all commit into one commit

git log --oneline
git reset --soft <#sha#>
git status
git commit --amend

Pull all branch from repo

git pull --all

Cherry pick

git cherry-pick <commit-hash>

Rebase

git rebase branch_name

Reverting rebase

git reflog
git log "HEAD@{5}"
git reset --hard "HEAD@{5}"

Remove branches no longer exist in remote

git fetch --prune

Rename branch local and repo

# Rename the local branch to the new name
git branch -m <old_name> <new_name>
	
# Delete the old branch on remote - where <remote> is, for example, origin
git push <remote> --delete old_name

# Push the new branch to remote
git push <remote> new_name

# Reset the upstream branch for the new_name local branch
git push <remote> -u new-name
